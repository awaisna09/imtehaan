# IMT Project

This is a web application for IMT (Imtehaan) - an educational platform.

## Project Structure

- Frontend: React-based web application
- Backend: Node.js/Express server
- Database: Supabase

## Features

- User authentication
- Chapter-based learning
- Topic-wise questions
- Progress tracking
- Score management
- Interactive lessons
- Chat functionality

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

## Technologies Used

- React
- Node.js
- Express
- Supabase
- Chart.js
- Webpack
